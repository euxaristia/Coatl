# Mee Roadmap

## v0.1 (now)
- Lexer + parser + AST
- Type checker for integers/bool
- WASM text emitter (WAT)
- Basic `fn`, `return`, integer arithmetic

## v0.2
- Ownership/borrow checking
- Structs, enums, pattern matching
- C FFI `extern`

## v0.3
- WASM binary emitter
- x86_64 SysV backend
- AArch64 backend

## v1.0
- Stable standard library
- Package manager
- Debug info + tooling
